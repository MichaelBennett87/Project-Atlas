# Portfolio Project Card Copy

## Title
Project Atlas

## Subtitle
AI-powered quantitative trading research platform

## Description
Project Atlas is a Java-based AI trading intelligence platform focused on real-time news analysis, market sentiment, momentum scoring, volume-first opportunity detection, and adaptive strategy improvement. The system integrates market data, catalyst classification, risk controls, trade journaling, and post-market performance review to explore how AI can support faster and more disciplined decision-making.

## Skills Highlighted
Java, Maven, REST APIs, WebSockets, AI/ML integration, ONNX Runtime, FinBERT, market data processing, risk logic, logging, software architecture
