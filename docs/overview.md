# Project Atlas Overview

Project Atlas is the flagship AI trading intelligence project in Michael Bennett's portfolio.

The purpose of this repository is to showcase the software architecture, engineering decisions, and ongoing development of a real-world AI-driven market analysis platform.

## Recruiter-Friendly Summary

Project Atlas demonstrates experience with:

- Java application development
- API integration
- WebSocket data streams
- AI/ML model integration
- Sentiment analysis
- Risk management logic
- Logging and observability
- Financial data processing
- Modular software architecture
- Iterative debugging and improvement

## Important Note

This project is best presented as a software engineering project, not as a promise of trading performance.
